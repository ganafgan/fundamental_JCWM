step by step membuat git

- buat folder repository di github
- init folder di komputer 	||=> git init ||(sekali)
- catet perubahan nya di stage 	||=> git add.(untuk semua) atau git add index.js(untuk spesifik file) || (sekali)
- catet perubahan di commit 	||=> git commit - "deskripsi"
- catet perubahan di github	||=> git push origin master


- inisialisasi origin		||=> git remote add origin link || sekali
- ubah link origin 		||=> git remote set-url link

- check log || git log --oneline
